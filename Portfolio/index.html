Projetos

Cifra de Cesar

O projeto consiste em implementar uma forma rudimentar de criptografia chamada Cifra de Cesar, contendo duas principais funcionalidades: criptografar e descriptografar uma mensagem. Na primeira etapa, o algoritmo irá ler um documento fornecido pelo usuário e será capaz de cifrar a mensagem, ou seja, aplicar o deslocamento das letras para direita ou esquerda, embaralhando a mensagem original. Logo após isso, ele armazena essa nova mensagem em um novo documento. Na segunda etapa, o algoritmo irá ler esse novo documento que foi gerado e mostrará qual é a mensagem original, desembaralhando a posição das letras.

Tecnologias utilizadas: Linguagem de programação Python

Sistema de sorveteria
